## Para iniciar el proyecto localmente

Clonar el repositorio

```bash
  git clone https://github.com/f-iturburu/Comision-23i-Proyecto-Modulo-4-Frontend
```


Instalar dependencias

```bash
  npm i
```

Iniciar el servidor

```bash
  npm run dev
```
